# SatoshiStream

## Overview
SatoshiStream is a comprehensive Bitcoin tracker project that enables users to monitor Bitcoin prices, transactions, and market trends in real-time.

## Features
- **Real-Time Tracking**: Monitor Bitcoin prices and market trends live.
- **Custom Alerts**: Set alerts for price changes and market movements.
- **User-Friendly Interface**: Simple and intuitive design for enhanced user experience.

## Installation
Follow these steps to install SatoshiStream:
1. Clone the repository:
   ```bash
   git clone https://github.com/mrfranklinisrael-ux/SatoshiStream.git
   ```
2. Navigate into the project directory:
   ```bash
   cd SatoshiStream
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
Access the tracker at `http://localhost:3000`.

## API Reference
- **GET /api/price**: Get the current Bitcoin price.
- **GET /api/transactions**: Fetch recent Bitcoin transactions.

## Customization
You can customize the application by modifying the configuration files located in the `config` directory.
